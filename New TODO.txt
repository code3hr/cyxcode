# POC Architecture

## Layer 1 — File Access Monitor

Intercept:

* readFile
* writeFile
* shell file operations

Track:

* path
* timestamp
* AI prompt correlation
* whether user explicitly requested access

---

## Example

User asks:

> “Fix my React app.”

AI accesses:

* `/project/src`

Normal.

But if AI accesses:

* `~/.ssh`
* browser cookies
* password stores
* unrelated documents

CyxWatch flags it.

---

# Layer 2 — Network Monitor

Critical feature.

Track:

* outbound requests
* domains
* upload size
* timing
* associated prompt/session

---

## Example Detection

User asks:

> “Summarize this code.”

AI sends:

* 20 MB compressed archive
* to unknown telemetry endpoint

CyxWatch alerts:

> “Unexpected data transmission detected.”

---

# Layer 3 — Prompt/Data Correlation

This becomes your secret weapon.

Correlate:

* user prompt
* accessed resources
* outbound traffic

Example:

| User Prompt      | AI Accessed           | Risk   |
| ---------------- | --------------------- | ------ |

# Intent-Aware Monitoring
